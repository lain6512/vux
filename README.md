# vuxFrame
Vue2.js+Bmob后端云+QQ授权登录,集成框架

### 启动

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9095
npm run dev

# build for production with minification
npm run build

```

### 包括功能
* QQ js-sdk授权；
* bmob后端云注册、登录；
* 切换页面左右滑入画出动画；
* 未登录判断和跳转，登录后返回之前页面；
* 初始化完成前的loading；

### 引入模块
* vue
* vue-router
* vuex
* vux
* axios
* webpack
